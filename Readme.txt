Utility scripts.

doctest - Utility to run doctests in a python script.

    $ doctest code.py

lightty - Script to run the given program as fastcgi using Lighttpd.

    $ lightty code.py
    http://0.0.0.0:8080
    ....
    $ lightty code.py 9000
    http://0.0.0.0:8080
    ....

pyvi - Script to open source file of a python module in vim.

    $ pyvi os.path
